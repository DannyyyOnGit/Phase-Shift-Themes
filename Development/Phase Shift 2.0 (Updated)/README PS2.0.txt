Theme Details
================
Phase Shift 2.0 Theme for Phase Shift (Steam Edition)
Created by: TheRealDannyyy
Bug Reports: https://github.com/TheRealDannyyy/Phase-Shift-Themes/issues

Installation
================
1. Put the "Phase Shift 2.0" folder inside Phase Shift "themes" folder (...Steam\SteamApps\common\Phase Shift\themes)
2. Enable "Menu video background" inside the options menu
3. Set "Neck" in profile options to "default"
4. Select "Phase Shift 2.0" theme inside options menu
5. Restart Phase Shift to reload and play

Requirements
================
- Phase Shift V1.38+ (Steam Edition)
- 16:9 widescreen monitor
- English language
- ~2gb available RAM
